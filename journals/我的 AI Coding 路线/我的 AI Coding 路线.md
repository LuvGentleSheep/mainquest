- # 前期
- 基于 chatgpt app 的 [👉 Work with Apps](https://help.openai.com/en/articles/10119604-work-with-apps-on-macos) 功能
	- 实现的项目有：
		- 众多 AppleScript 和快捷指令工具 for iOS & macOS
		- [👉 Kelvinstyle](https://kelvinstyle.netlify.app/homepage/homepage_index.html)
		- [👉 Pikara](https://apps.apple.com/us/app/pikara/id6747956457) for iOS
		- [👉 Snapatch](https://apps.apple.com/us/app/snapatch/id6748052970) for iOS
	- 优势：可以和 macOS 上的文本编辑工具、终端和主流 IDE 协作，无需复杂配置
	- 劣势：稳定性差，丢失对文本、窗口的识别是常有的事，只能追踪单个窗口
- # 当下
- 基于 Cursor 和 Codex 的更快捷高效的 Vibe Coding
	- 实现的项目有：
		- 诞生于 work with apps 的项目的后期优化
	- 优势：无需 OpenAI 的 API，可以直接通过登陆 OpenAI 访问使用 Codex 账号。无缝接管工程项目，可以同时读写多个文件，稳定性极强
	- 劣势：容易触及 Codex 的使用上限，如果是从零构建项目，很快就会花光当前时间段限额
